# 全新重写 Flutter TVBox 项目
> 这是一套完全从零重写的完整开源Flutter TVBox项目，所有源码100%公开可修改，没有任何加密或隐藏内容。

## 项目说明
本项目是为你全新重写的纯Flutter跨平台TVBox应用，完全满足你的需求：
1.  **仅保留核心功能**：只保留了影视播放与弹幕功能，完全移除了小说、漫画等无关功能
2.  **完整源支持**：整合了PeekPili的`drpys`源解析模块，完整支持所有标准TVBox源格式（t3/t4/drpy源）
3.  **弹幕功能**：完整保留了弹幕模块，支持弹幕调节、过滤等功能
4.  **免证书打包**：内置了完整的免Apple开发者证书打包配置，打包出的IPA支持巨魔商店、爱思助手自签安装

## 功能说明
- ✅ 支持TVBox标准影视源配置
- ✅ 支持影视分类、列表展示
- ✅ 支持视频播放，全平台兼容
- ✅ 支持弹幕显示与控制
- ✅ 支持源配置本地保存
- ✅ 完整的免证书自动打包流程

## 打包与安装
### GitHub Actions 自动打包
项目已经内置了完整的自动打包工作流，你只需要：
1.  将本项目推送到你的GitHub仓库
2.  进入Actions页面，手动触发`Build for iOS`工作流
3.  等待打包完成，下载未签名IPA，用巨魔商店/爱思助手自签安装即可

### 本地打包
你也可以使用内置的本地打包脚本，在本地完成免签名打包：
```bash
chmod +x package_ios.sh
./package_ios.sh
```

## 使用说明
1.  打开应用后，点击右上角的设置按钮，配置你的TVBox影视源地址
2.  配置完成后，应用会自动加载源的影视内容
3.  点击影视条目，即可进入播放页面，自动加载弹幕
4.  打包后的IPA，用你熟悉的自签工具安装到iPhone即可
